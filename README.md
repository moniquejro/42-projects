# 42 Projects

Este repositório reúne os projetos que desenvolvo durante minha formação na 42.

A 42 é uma escola de programação com metodologia baseada em aprendizado prático, resolução de problemas e colaboração entre alunos, sem aulas tradicionais. Cada projeto propõe desafios que desenvolvem fundamentos sólidos de programação, autonomia e pensamento lógico.

Aqui, organizo meus projetos conforme avanço na trilha da 42, além de revisar e aprimorar soluções com o tempo.
